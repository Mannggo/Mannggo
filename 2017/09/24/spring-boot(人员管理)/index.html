IDEA开发spring-boot项目(人员管理)

创建工程

  1. file -> new project
    选择你的JDK版本，最好是Java1.8，点击next

  1. 调整Java版本，选择构建Maven项目，next

  2. 选择需要的起始依赖，spring-boot会帮你自动添加和下载依赖（Maven）,我在这只选了web和Thymeleaf,web会提供tomcat支持和SpringMVC，Thymeleaf则提供视图的模板引擎。不建议使用jsp。next。

  3. 确定项目名称和项目位置，Finish。

项目结构

生成的项目结构如下图所示：

目录结构与maven项目一样，不过自动生成了一个DemoApplication类，这个类是项目的引导程序，启动这个类中的main方法即可启动spring-boot项目。
