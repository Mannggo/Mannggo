Redis数据操作——（1）String类型相关操作

String

一个键对应一个字符串，一个键最多能存储一个512Mb的String

相关操作

设置操作

1. set key value
2. setnx key value
3. setex key 超时时间 value
4. mset key value [key] [value] …
5. msetnx key value [key] [value] …

获取操作

1.get key
2 .mget key [key]…
3.getrange key 开始下标 结束下标

修改操作

1.getset key value
2.append key value
3.setrange key 偏移量 value
4.incr key
5.incrby key 增长步长
6.decr key
7.decrby key 递减步长

删除、查看长度

1.del key [key]
2.strlen key
